Тема: Структура HTML-документа

Мета: Ознайомитись із базовою структурою HTML-документа, навчитися створювати елементи сторінки, таблиці, форми та вставляти зображення.

Місце розташування: GitHub

Опис предметного середовища

Розробка вебзастосунку для онлайн-платформи CyberZone, що представляє комп’ютерний клуб нового покоління з можливістю перегляду послуг, галереї, контактної інформації та бронювання місць онлайн. Мета проєкту — створити зручний вебсайт, який дозволяє користувачам дізнатися про клуб, його обладнання, доступні турніри та VR-зону, переглядати фото інтер’єру та ігрових станцій, а також бронювати місця для гри або участі в турнірах. Ціль — підвищити залученість геймерів, спростити процес бронювання і створити інтерактивний портал для спільноти CyberZone. Сценарій використання сайту: Користувач заходить на сайт і бачить головну сторінку з великим блоком-привітання, який підкреслює сучасність клубу та його можливості. Далі він може: Перейти в розділ «Про нас», щоб дізнатися про історію клубу, оснащення та атмосферу. Ознайомитися з розділом «Послуги», де представлені: ігрові місця, турніри та VR-зона. Переглянути галерею, яка демонструє фото клубу, ігрових станцій та VR-обладнання. Використати форму бронювання, щоб обрати дату та час для відвідування клубу. Зареєстрований користувач (або адміністратор сайту) може: Переглядати всі заброньовані місця (у майбутньому можна додати інтеграцію з базою даних). Керувати інформацією про клуб та додавати нові події або турніри. Сайт забезпечує зручну навігацію, сучасний дизайн у кіберпанк-стилі, інтерактивні елементи (попапи та lightbox для галереї) та адаптивність для різних пристроїв, що робить його зручним як для новачків, так і для постійних відвідувачів CyberZone..

Структура документа

<!DOCTYPE html>
<html>
  <head>
    <title>CyberZone — Комп'ютерний клуб</title>
  </head>
  <body>
    <header>Заголовок сайту</header>
    <nav>Меню</nav>
    <main>Основний контент</main>
    <footer>Підвал сторінки</footer>
  </body>
</html>

HTML-код форми

  <input id="name" type="text" placeholder="Введіть ваше ім’я"><br />

  <label for="email">Email:</label><br />
  <input id="email" type="email" placeholder="example@gmail.com"><br />

  <label for="message">Повідомлення:</label><br />
  <textarea id="message" placeholder="Ваш текст..."></textarea><br />

  <button>Надіслати</button>
