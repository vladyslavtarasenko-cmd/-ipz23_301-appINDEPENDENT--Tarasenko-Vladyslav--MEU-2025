Тема та мета роботи

Тема: Каскадні таблиці стилів. Селектори. Ідентифікатори. Стильове оформлення текстових елементів.

Мета: Придбати практичні навички роботи з селекторами тегів, класів, ідентифікаторів, списками, властивостями кольору, фону, оформленням текстових елементів у HTML-документах.

Місце розташування: GitHub

1. Селектори тегу

Селектори тегів застосовуються безпосередньо до всіх елементів одного типу.

 body { 
  margin: 0;
  font-family: 'Poppins', sans-serif;
  background-color: #F0EADB;
}

Це приклад тексту, оформленого через селектор тегу body.

2. Селектори класу

Клас дозволяє задавати унікальне оформлення для групи елементів.

.hero {
  height: 100vh; /* висота на весь екран */
  background: url('https://images.unsplash.com/photo-1603791452906-bcbf7edb9b0d?auto=format&fit=crop&w=1600&q=80') center/cover no-repeat; /* фонова картинка */
  display: flex; /* флекс-контейнер */
  justify-content: center; /* по горизонталі по центру */
  align-items: center; /* по вертикалі по центру */
  text-align: center; /* центр тексту */
}

Цей текст оформлено за допомогою класу .hero.

3. Селектори ідентифікатора

Ідентифікатор використовується для одного унікального елемента сторінки. Позначається знаком решітки #.

#toggle-search {
  display: none;
}

Цей абзац має унікальний стиль через ID-селектор.

4. Інші селектори

Можна використовувати дочірні, сусідні, атрибутні та універсальні селектори:

.about, .services, .contact {
  text-align: center; /* центр тексту */
  padding: 100px 20px; /* внутрішні відступи */
  max-width: 900px; /* максимальна ширина контенту */
  margin: 0 auto; /* центрування по горизонталі */
}

Висновки

У ході виконання лабораторної роботи я закріпив знання про роботу з CSS-селекторами, навчилвся застосовувати стилі до елементів сторінки, оформлювати текст, таблиці та списки. Також ознайомився з принципами каскадування та спадкування стилів.

Контрольні питання

  1. Для чого використовуються каскадні таблиці стилів?
      2. Каскадні таблиці стилів (CSS) використовуються для оформлення та візуального представлення HTML-елементів на вебсторінці — тобто, щоб задавати кольори, шрифти, розміри, розташування та інші стилі.
  2. Які методи визначення стилів у HTML-документі ви знаєте?
    • Зовнішній — через окремий CSS-файл, підключений за допомогою тегу 
      <link rel="stylesheet" href="style.css">
    • Внутрішній — у самому HTML-документі всередині тегу
      • <style>.../style>
    • Вбудований (inline) — безпосередньо в атрибуті елемента 
      <p style="color: green;">Текст
      >
  3. У чому різниця між принципами каскадування і спадкування?
    • Каскадування — це принцип, за яким, якщо до одного елемента застосовано кілька стилів, браузер вибирає той, що має вищий пріоритет (за специфічністю або порядком підключення).
    • Спадкування — це принцип, за яким деякі властивості (наприклад, колір або шрифт) автоматично передаються дочірнім елементам від батьківських.
  4. Як оформлюються селектори тегу і класу?
      5. Селектор тегу оформлюється просто назвою тегу: 
    • p { color: blue; }
      • Селектор класу оформлюється з крапкою перед назвою класу: 
    • .text { color: blue; }
  5. Для чого використовується ідентифікатор?
      6. Ідентифікатор (атрибут id) використовується для унікального позначення елемента на сторінці, щоб до нього можна було звертатися в CSS, JavaScript або посиланнях.
  6. Навести приклади оформлення текстових елементів.
      7.       p {
  color: #333;              /* колір тексту */
  font-size: 18px;          /* розмір шрифту */
  font-family: 'Poppins', sans-serif; /* тип шрифту */
  text-align: center;       /* вирівнювання тексту */
  font-weight: bold;        /* товщина шрифту */
  text-decoration: underline; /* підкреслення */
  text-transform: uppercase; /* усі літери великі */
  line-height: 1.6;         /* міжрядковий інтервал */
}
